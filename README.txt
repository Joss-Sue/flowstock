usar de ejemplo categorias, puedes seguir este orden o el inverso
-> index.js(agregar la ruta) 
-> routes(copiar categoria-routes y modificarlo para que coincida con el nombre) 
-> controllers (copiar categoria_controller.js y modificar los nombres)
-> models (copiar mongoose/categoria-model.js y modificar los nombres)
-> env (renombra el env sin ejemplo y agrega tu ruta de mongo)
-> instala extension REST Client para probar cada api completa(tambien se puede usar postman)
